# PRODIGY_SD_03
📞 Simple Contact Management System

A web-based Contact Management System built using Python and Streamlit that allows users to store and manage contact information efficiently.

This application supports adding, viewing, editing, and deleting contacts with persistent storage using a CSV file.


🚀 Features

✅ Add new contacts (Name, Phone, Email)

✅ View all saved contacts

✅ Edit existing contacts

✅ Delete contacts

✅ Data stored in CSV file (persistent storage)

✅ Clean and interactive Streamlit UI

✅ Deployable on Streamlit Cloud


🛠️ Tech Stack

Python

Streamlit

Pandas

CSV File Storage
